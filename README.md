# jCal.js
##Jewish and Gregorian event calendar - jQuery plugin

##Features

- Fully jewish abd gregorian calendar.
- Event list detail view mode.
- Add events (xml, json or ajax-php).
- Responsive design.
- Basic views or jQuery-ui theme.
- and more...
